# Kubectl

O **kubectl** é um comando que utilizamos para nos comunicar com o APIServer do Kubernetes. Com ele nós podemos, entre outras coisas

- Atualizar
- Vizualizar
- Gerenciar

Tanto Pods, Services, Deploymets e etc.

É interessante ler posteriormente a documentação do Kubernetes sobre o **kubectl**

[Link Para Documentação Kubectl](https://kubernetes.io/docs/reference/kubectl/)

# Comandos Diários

### - kubectl apply -f arquivo.yaml

- kubectl get pods

- kubectl portfoward

- kubectl delete -f e variantes
